# Lessons Learned

This project helped me gain hands-on experience with:

- Amazon S3 Static Website Hosting
- Amazon CloudFront CDN
- DNS Management with Route 53
- HTTPS using ACM
- Origin Access Control (OAC)
- AWS Networking
- Secure Cloud Architecture
- Global Content Delivery
- Low-cost website hosting

One of my biggest takeaways was realizing that building a website isn't just about uploading files. It's about designing an architecture that is secure, reliable, scalable, and delivers a great experience for users around the world.
