# Estimated Monthly Cost

| AWS Service | Estimated Cost |
|--------------|----------------|
| Amazon S3 | Few cents/month |
| CloudFront | ~$0.60 per 1 million requests |
| Route 53 Hosted Zone | $0.50/month |
| AWS Certificate Manager | Free |
| Domain Registration | $1.1/year |

For a personal portfolio website, the total monthly cost is typically less than **$5 - 7**, depending on traffic.
